## Cara Run
1. Jalankan nodemon:
   ```
   npx nodemon -e html,js,css
   ```
   atau,
   ```
   npm run start-dev
   ```
2. Jalankan emulator firebase:
   ```
   firebase emulators:start --import=emulatorData  --export-on-exit=emulatorData
   ```
   atau,
   ```
   npm run fb-emu
3. Buka web di http://localhost:5000 .
